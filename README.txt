INTRODUCTION
 ************ 
Provides a Point of Sale Athorized.net payment getway support with card swipe
for Drupal Commerce POS system, allowing.


REQUIREMENTS
 ************ 
This module requires the following modules:

 *Drupal Commerce (https://drupal.org/project/commerce)
 *Commerce Authorize.Net Card Present

  (https://www.drupal.org/project/commerce_authnet_card_present)

 *Commerce Point of Sale (https://www.drupal.org/project/commerce_pos)

 *jQuery Update (https://www.drupal.org/project/jquery_update) running jQuery
1.7 or higher.

INSTALLATION
 ************ 
 *Install as you would normally install a contributed Drupal module.
See:

https://drupal.org/documentation/install/modules*themes/modules*7
for further information.

CONFIGURATION
 ********************* 

Configure settings in
Home >> Administration >> Store >> Configuration >> Payment methode
 >> Authorize.Net AIM * Card Present >> Editing reaction
rule "Authorize.Net AIM * Card Present"

Before use this module please configure the Getaway setting.
